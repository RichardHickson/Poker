Poker Ledger website which enables login/sign up. Allows you to set starting buy in, chips while keeping track of total chips and re-buys.
At the end of a game, profit and cashout is calculated for easy cashing out by a designated banker.
A poker game can be stored in a database allowing for checking past games and how well a person does over time.
The site is hosted on Azure and the link is: https://ambitious-forest-0e1870710.3.azurestaticapps.net/
